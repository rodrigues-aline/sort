## Get started

virtual enviroment instalation (linux - python3)
```
sudo apt install python3.8-venv
```

create enviroment
```
python3 -m venv venv
```

activate enviroment
```
source venv/bin/activate
```

install dependencies
```
pip3 install -r requirements.txt
```

run project
```
python main.py
```
